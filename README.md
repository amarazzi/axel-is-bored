# axel is bored

My personal website. A place where I share what I'm working on, what I'm writing, and the small apps I build when I get bored.

Built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and TypeScript.

## What's inside

- **Home** — A quick intro, recent posts from my Substack ([observando](https://observando.substack.com)), and a preview of my projects.
- **Observando** — Essays about how I see life. Written in Spanish.
- **Projects** — Small apps I vibescode and design: [1oo1](https://github.com/amarazzi/1oo1), [minimal](https://github.com/amarazzi/minimal), and whatever comes next.
- **About** — A bit about me.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it.
